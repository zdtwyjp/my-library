SuperMap iClient 6R(2012) for JavaScript


SuperMap iClient 6R(2012) for JavaScript 是一套由 JavaScript 语言编写的 GIS 客户端应用开发包， 支持多源数据地图，支持多终端，跨浏览器， 通过本产品可快速实现浏览器上美观、流畅的地图呈现。

产品变更信息

当前版本：6.1 SP1
上次版本：6.1.0

产品提供了地图图层、地图控件、专题图、查询、量算、空间分析、网络分析、数据集查询、交通换乘、UGC图层服务等功能， 较之前版本本次改版的产品功能更加丰富、性能 更加优越、扩展性更强、 兼容效果更好，本次版本主要做了以下改进和优化：

  • 1. 新增功能：
    • (1) 新增动态分段服务，仅支持 SuperMap iServer 6.1.2 及其以上版本的动态分段服务
    • (2) 新增数据服务，提供 ID 查询、几何查询、缓冲区查询、SQL 查询、地物编辑等功能
    • (3) 新增 Canvas2 渲染方式，适用于制作矢量数据的动画渲染
    • (4) 新增离线存储和访问功能，支持基于 PhoneGap 开发的移动 Web 应用， 实现离线状态下的地图存储与访问
    • (5) 新增交通换乘服务，仅支持 SuperMap iServer 6.1.2 及其以上版本的交通换乘服务
    • (6) 新增 UGC 图层服务，包括 UGC 栅格图层，UGC 矢量图层，UGC 影像图层，UGC 专题图图层
    • (7) 新增空间关系分析服务，仅支持 SuperMap iServer 6.1.2 及其以上版本的空间关系服务
  • 2. 接口变更：
    • (1) CanvasLayer 类新增 dpi 属性
    • (2) SampleCacheLayer 类删除了viewBounds、viewer、format三个属性
    • (3) CloudLayer 类新增 url 属性
    • (4) PathGuideItem 类新增 geometry 和 description 属性
    • (5) SuperMap.Feature.Vector 中 attributes 属性修改为键值对形式
    • (6) SuperMap.Control.PanZoomBar 中新增 showSlider 属性
    • (7) SuperMap.Control.PanZoomBar 新增 levelsDesc 属性
    • (8) SuperMap.Layer.TiledDynamicRESTLayer 新增 dpi 和 projection 属性，同时关闭 viewBounds 、viewer、scale 三个属性， 删除设置在 url上的 maxVisibleVertex 参数
    • (9) 删除 Ajax 类
    • (10) SuperMap.REST.ThemeParameters 新增 displayFilter 和 displayOrderBy 属性
    • (11) SuperMap.REST.TransportationAnalystParameter 新增 barrierPoints 属性
  • 3. 性能优化：
    • (1) 缩放动画与平移交互操作优化
    • (2) 瓦片渐进效果优化
    • (3) 多图层叠加性能优化
    • (4) 动画重构与插件化
  • 4. 产品包以及产品化工作的完善：
    • (1) 统一命名空间为 SuperMap
    • (2) 精简库文件，压缩核心库
    • (3) 全新的产品页面样式，对产品页面整体的内容结构、页面布局做了重大调整
    • (4) 帮助文档本地化
    • (5) 控件及其样式的调整，部分控件进行了功能完善和改进，删除了 Pan， PanPanel， ZoomIn， ZoomOut， ZoomPanel， ZoomToMavExtent 等控件， 将 PanZoom 合并到 PanZoomBar，并且优化了 PanZoomBar
  • 5. 新增范例：
    • (1) 动态分段专题范例
    • (2) 数据集缓冲区查询范例
    • (3) 数据集 SQL 查询范例
    • (4) 数据集几何查询范例
    • (5) 数据集 ID 查询范例
    • (6) 高性能矢量图层渲染---车辆监控系统范例，须在支持 HTML5 Canvas 的浏览器下运行
    • (7) 高性能矢量图层渲染---大数据矢量地物查询与交互范例，须在支持 HTML5 Canvas 的浏览器下运行
    • (8) 高性能矢量图层渲染---气象监测范例，须在支持 HTML5 Canvas 的浏览器下运行
    • (9) WMTS 图层使用范例
    • (10) 新增交通换乘范例
  • 6. 修复的问题：
    • (1) 修复地图控件设置固定比例尺后，PanZoomBar 控件显示异常问题
    • (2) 修复 WMTS 图层出图问题
    • (3) 更新天地图图片地址
    • (4) 修复使用 map 获取比例尺出错问题
    • (5) 解决多点量算、跨域量算问题
    • (6) 修复加载 4.0 + 缓存的出图错误问题
